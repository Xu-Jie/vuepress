## 网吧管理

::: tip
如要进行远程控制需要先下载远控插件

远程断开的时候会有提示弹窗一直停留在头部，但是需要手动关闭
:::

## 员工管理

::: tip
员工管理的所有操作，需要先验证你的短信，验证过一次后就不需要验证了，防止盗号操作
:::

## 其他

::: tip
1.switch函数注意补充default

2.setInterval替换为setTimeout

3.方法名以及参数名注意语义化，不可使用one,two,three,a,b,c...
:::
