## 摇晃效果

> 调用方法

```js
  this.$domFn.shake(dom)
```

----------

## 监听

> 调用方法

```js
  this.$domFn.addListen(dom, event, fn)
```

## 动画特效

> 调用方法

```js
  this.$domFn.addAnimateClass(dom, animateClass)
```

## 添加class

> 调用方法

```js
  this.$domFn.addClass(dom, cls)
```

## 移除class

> 调用方法

```js
  this.$domFn.removeClass(dom, cls)
```
